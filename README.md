# WhatsApp Bot com Python

Bot simples para automação de envio de mensagens e imagens no WhatsAppWeb.

## Funcionalidades

* Envio automático de mensagens
* Envio de imagens
* Interface gráfica com Tkinter
* Controle de iniciar/parar envio

## Tecnologias

* Python 3
* Tkinter
* PyAutoGUI
* Pyperclip

## Instalação

```bash
pip install -r requirements.txt
```

## Como executar

```bash
python interface.py
```

## Estrutura do projeto

```text
.
├── assets/
├── bot.py
├── interface.py
├── grupos_exemplo.txt
├── requirements.txt
└── README.md
```

## Observações

* O WhatsAppWeb deve estar aberto.
* Ajuste as imagens de referência conforme sua resolução.
* O arquivo `grupos_exemplos.txt` contém exemplo de grupos.
